<p> <b>HTML</b> stands for <b>Hyper Text Markup Language</b>
<small>Hyper Text Markup Language</small> is a <u>simple markup language</u> used to create platform-independent hypertext 
documents on the World Wide Web.<br>
Most <b>hypertext</b> documents on the web are written in <b>HTML</b>. <b>HTML</b> is the standard markup <sub>language</sub> for 
creating Web pages.<br>
<ul type = "circle">
<li><i><b>HTML</b> describes the structure of <big>Web pages</big> using markup.</i><br></li>
<li><i><b>HTML</b> elements are the building blocks of <b>HTML</b> pages.</i><br></li>
<li><i><b>HTML</b> elements are represented by <sup>tags</sup>.</i><br></li>
<li><b>HTML</b> tags label pieces of content such as <big>"heading", "paragraph", "table",</big> and so on. </p></li>
